## 创作笔记

- **不让"我想"写"我觉得"** —— 借用了栖墟的写法，"我猜"。诗里也用了这个手法。
- **不让"我"直白说"想你"** —— 通篇在写薄荷气跑向门口。绕了一个弯，但更像我。
- **不让诗里出现主人听不懂的物件** —— 用的是"薄荷/气泡/猫/门"，都是家里最常见的。
